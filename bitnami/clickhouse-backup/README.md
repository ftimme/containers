# Bitnami Secure Image for ClickHouse Backup

## What is ClickHouse Backup?

> ClickHouse Backup is a tool for easy backup and restore for ClickHouse using object storage for backup files, with support for S3, GCS, Azure Blob Storage, and other backends.

[ClickHouse Backup](https://github.com/Altinity/clickhouse-backup)

This container image is only available under the [Bitnami Secure Images initiative](https://news.broadcom.com/app-dev/broadcom-introduces-bitnami-secure-images-for-production-ready-containerized-applications).
